## KuramotoCoupling

### Coupling Equation

$$c_i = \frac{a \sum_{j=0}^{-1 + N} {w}_{i,j} \sin{\left(- x_{i} + x_{j} \right)}}{N}$$

### Properties

- **Delayed:** Yes
- **Sparse:** No

### Parameters

| Name | Default | Description |
|------|---------|-------------|
| $a$ | 1.0 | Rescales the connection strength. |
| $N$ | 1.0 | Number of nodes, i.e., size of network. |
